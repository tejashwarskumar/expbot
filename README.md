# Exp Bot Demo Documentation
## App location: [Exp Bot · Streamlit](url)
## Bot interface:
![image](https://github.com/user-attachments/assets/21185b2a-459e-4996-b7ea-57df223bb322)
### The bot has 2 modules which are the “Advisor” module and the “Query Experiments” module. The Advisor module answers questions which are semantic in nature and require the user to ask general questions regarding experimentation.
### The Query Experiments module answers questions which are related to the list of experiments in the backend for which the information has been compiled.
